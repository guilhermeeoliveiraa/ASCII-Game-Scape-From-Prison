# ASCII-Game-Scape-From-Prison

An ASCII terminal game where the player must escape from a prison by navigating through a map, avoiding obstacles, and finding the exit.

The project was developed with focus on **structured / object-oriented programming**, **terminal input handling**, **game logic**, and **state management**.

---

# 🎮 About the Game

**Escape From Prison** is a simple terminal-based game where the player controls a character inside a prison map represented using ASCII characters.

The main objectives are:

- Explore the prison map
- Find the exit
- Avoid obstacles and blocked areas
- Defeat or evade enemies
- Discover the correct path to escape

The entire game runs inside the **terminal**, using ASCII characters to render the environment.
